Rock Paper Scissors Draw 4

  • High Concept: 3 Players take turns placing marks on a grid like in tic tac toe, the first to draw 4 in a row wins, but players can each delete each other marks.
  • Theme: Special dispensation to use my previous analog game
  • Mandated Variety: Clickable Buttons, Deterministic, Competitor, Puzzle
  • Player Experience Goals: Quick and easy to play game when with friends
  • Gameplay: Each player takes turns placing a Rock/Paper/Scissor mark on the grid, the first player to draw 4 in a row wins.  As a twist if a mark get surrounded on 2 sides by a mark that deletes it (for example paper gets surrounded by scissors) then the mark is deleted and removed.  In addition a mark can be protected if it is surrounded by at least 2 of the same mark.
  • Prototype Goal: Make sure all grid mechanics work as intended.  Make sure the system can search the grid and detect stuff like 4 in a row, and when a mark is surrounded.  The next iteration would hopefully have a smoother clickable interface and animations showing when marks are safe/deleted
  • Features: Simple rules, but allow some degree of 3 player compitition
  • Setting/Premise: No real setting at the moment, but I do think the theme could work with some sort of stratagy game involving placing castles/soldiers.
  • Story:  No story at the moment, this game is intended for compitition amoung friends.
  • Target Audience: 3 friends who want a quick game to play together
  • Play Time: around 1 minute to 2 minutes, depending on play style
  • Strategies: Trying to create a box of 2x2 of one mark often gives you an advantage I've found
  • Prototype Assets: All art was made by me (should be obvious from the very simple design).  No other assets used.  Though the second iteration might include some sound effects for deleting and protecting marks
  • Prototype Closest Other Game: No particular closest game other then a modified version of tic tac toe
